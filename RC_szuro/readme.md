# Rc szűrő

Szül.:2006.05.04.

Aluláteresztő RC szűrő kapcs.

*Adatok:*

$R=5O Ohm$

$C=4 uF$

<img width="818" height="588" alt="image" src="https://github.com/user-attachments/assets/35b99291-2e79-489a-8f09-4c6ea22ae139" />

*Határfrekvencia:*

$fh= {1 \over (2 * Pi * R * C)} $   
$fh= {1 \over (2 * Pi * 5R * 4uF)} $  
$fh=7957.7471Hz$
